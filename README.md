# Leanding Page "GoIT"

Team project on HTML5, SCSS

## Requirements
- HTML5
- SCSS
- Gulp
## Installation
```bash
git clone https://github.com/OlegGap/best_team_project
```
```bash
cd ./best_team_project
```
```bash
npm install
```
## Development and Deployment
```bash
npm start #Start development
```
```bash
npm run build #Build to production
```
